# Scorciatoie

**Barre laterali flottanti di scorciatoie rapide per Windows**

Un tool leggero e personalizzabile per avere sempre a portata di mouse le tue applicazioni, cartelle, testi fissi e clipboard multipli.

![Screenshot principale](screenshot-main.png) <!-- aggiungerai lo screenshot dopo -->

## ✨ Funzionalità principali

- Barre laterali **flottanti** e trascinabili (una per gruppo)
- Scorciatoie con icone personalizzate
- **Multi-Clipboard** avanzato
- Testi fissi rapidi
- Gestione gruppi e colori
- Salvataggio automatico della configurazione
- Drag & Drop per riordinare le scorciatoie
- Supporto multi-monitor

## 🚀 Come compilare

1. Apri la soluzione `Scorciatoie.sln` con Visual Studio 2022 o superiore
2. Compila (F5)
3. L’eseguibile si trova in `bin\Debug\net6.0-windows`
